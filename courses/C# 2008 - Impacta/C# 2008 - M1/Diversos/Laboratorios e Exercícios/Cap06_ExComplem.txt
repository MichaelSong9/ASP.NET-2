Exercícios Complementares

1. Escreva o código necessário para criar, percorrer e exibir em um messageBox os elementos de um array de números inteiros chamado "Numeros".
Resposta;
int[] Numeros={7,3,4,1,67,-2,3};
foreach(int numero in Numeros)
{
	MessageBox.Show(numero);
}

2. Escreva o código necessário para localizar, em um array de nomes chamado "Lista", o nome "José".
Resposta:
if (Lista.IndexOf("José")>=0)
{
	MessageBox.Show(Lista[ Lista.IndexOf("José")]);
}
