2. Qual é o controle mais adequado para se obter uma data digitada pelo usuário

3. Escreve o código necessário para exibir uma caixa de diálogo que obtenha uma cor escolhida pelo usuário e a coloque na cor de fundo do formulário

{
	this.backcolor=cd.color
}
